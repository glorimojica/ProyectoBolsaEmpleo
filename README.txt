
PROYECTO: BOLSA DE EMPLEO

Descripción:
Sistema web desarrollado en Java, Spring Boot y MySQL para gestionar una bolsa de empleo. Permite registrar empresas, oferentes, puestos, habilidades y generar reportes.

Base de datos:
Nombre: bolsa_empleo

Antes de ejecutar:
1. Ejecutar el script SQL entregado
2. Revisar application.properties
3. Verificar usuario y contraseña de MySQL

http://localhost:8080

http://localhost:8080/login

Funcionalidades:
- Parte pública: ver puestos, buscar puestos, registro
- Empresa: crear puestos, agregar requisitos, ver candidatos
- Oferente: registrar habilidades, subir CV
- Administrador: aprobar registros, gestionar características, generar PDF
